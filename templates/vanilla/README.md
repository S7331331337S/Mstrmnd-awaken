# __APP_TITLE__

A lightweight web starter generated by the Mstrmnd CLI. It is deploy-ready for Vercel and uses a single HTML/CSS/JS bundle so you can move fast without extra tooling.

## Commands

- `npm run dev` – serves the site locally via `npx serve .`
- `npm run start` – alias for `dev`

## Deploy

If you included `vercel.json`, deploy with:

```bash
vercel deploy
```

Otherwise, add `--vercel` when running `mstrmnd init` to generate the config automatically.
